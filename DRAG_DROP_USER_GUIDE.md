# 🎯 **Drag-Drop Section Manager - Complete User Guide**

## 🚀 **How to Use - Step by Step**

### **1. Access the Feature**
- Go to: **Resume Builder** (`/dashboard/job-seeker/resume-builder`)
- Look for the **"Resume Section Manager"** at the top
- You'll see all resume sections in a beautiful interface

### **2. Edit Mode (Default)**
In Edit Mode, you can:
- **Drag sections** to reorder them
- **Lock/unlock sections** 
- **Show/hide sections**
- **Move sections** with arrows
- **Reset to default order**

### **3. Preview Mode**
In Preview Mode, you can:
- **See numbered badges** showing section order
- **View actual sections** in your custom order
- **Hidden sections** are completely omitted
- **Real-time updates** as you make changes

---

## 🎮 **Interactive Controls**

### **🖱️ Drag and Drop**
1. **Click and hold** the grip icon (≡) on any section
2. **Drag** it up or down to reorder
3. **Release** to drop in new position
4. **Visual feedback** shows where it will drop

### **🔒 Lock/Unlock Sections**
- **Click lock icon** (🔒) to prevent movement
- **Locked sections** have gray background
- **Personal Info** is locked by default (always first)
- **Unlock** to allow reordering

### **👁️ Show/Hide Sections**
- **Click eye icon** (👁️) to toggle visibility
- **Visible** = green eye icon
- **Hidden** = gray eye icon
- **Hidden sections** won't appear in final resume

### **⬆️ Move Up/Down**
- **Click up/down arrows** for precise positioning
- **Disabled** at boundaries (can't move first up, last down)
- **Alternative** to drag-and-drop
- **Perfect for mobile** or precise adjustments

### **🔄 Reset to Default**
- **Click "Reset Order"** to restore professional ordering
- **Returns** to ATS-optimized section sequence
- **Useful** if you make too many changes

### **Section Badges**
- **#1, #2, #3...** - Shows current order

### **Status Icons**
- **🔒 Locked** = Cannot be moved
- **👁️ Visible** = Section will appear
- **👁️‍🗨️ Hidden** = Section won't appear
- **≡ Grip** = Drag to reorder

### **Section Colors**
- **🔵 Blue**: Personal Information (critical)
- **🟢 Green**: Professional Summary (important)
- **🟣 Purple**: Experience (most important)

- **🟠 Orange**: Education (background)
- **🔴 Red**: Certifications/Awards (optional)

## 🎯 **Best Practices**

### **ATS Optimization**
1. **Personal Information** - Always first (locked)
2. **Professional Summary** - Second (important)
3. **Experience** - Third (most critical)
4. **Skills** - Fourth (technical roles)
5. **Education** - Fifth (background)
6. **Projects** - Sixth (portfolio)
7. **Certifications** - Seventh (optional)
8. **Awards** - Eighth (optional)

### **Customization Tips**
- **Tech roles**: Move Skills higher
- **Executive roles**: Move Summary higher
- **Academic roles**: Move Education higher
- **Creative roles**: Move Projects higher

### **When to Lock Sections**
- **Personal Info**: Always locked (ATS requirement)
- **Summary**: Lock if critical for your industry
- **Experience**: Lock if you want to maintain order
- **Skills**: Unlock for flexibility

## 🔧 **Troubleshooting**

### **Drag Not Working?**
1. **Check if section is locked** (gray background)
2. **Try the up/down arrows** instead
3. **Refresh the page** and try again
4. **Check browser compatibility**

### **Buttons Not Responding?**
1. **Click directly on icons** (not text)
2. **Wait for animations** to complete
3. **Check if section is locked**
4. **Try keyboard navigation**

### **Changes Not Saving?**
1. **Check internet connection**
2. **Wait a few seconds** for auto-save
3. **Manually save** the resume
4. **Refresh** and check again
